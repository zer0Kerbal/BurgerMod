# BurgerMod! (BURG) 

All of the layers are separate components, with stats, that can be stacked to make the gigantic Jeb-snack of your dreams, and fly it. For Kerbal Space Program.

## Description 

## Features 

This mod adds eleven parts
- Heel (Bun Bottom)
- Meat
- Cheese
- Lettuce
- Tomato
- Onion
- Pickles
- Ketchup
- Crown (Bun Top)
- Bacon wings
- Bacon Winglets
